# Password Strength Analyzer

## Overview

Password Strength Analyzer is a full-stack cybersecurity web application that evaluates password security and provides recommendations for creating stronger passwords.

The application analyzes passwords based on common security best practices and helps users identify weaknesses before using a password.

---

## Features

- Password strength scoring
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Detects common weak passwords
- Detects repeated characters
- Detects common attack patterns
- Provides security recommendations in real time

## How It Works

The user enters a password into the application.

The backend analyzes:

- Password length
- Character variety
- Common password usage
- Repeated characters
- Predictable patterns

The application then returns:

- Strength rating (Weak, Moderate, Strong)
- Security score
- Personalized recommendations

## Installation

Clone the repository:

```bash
git clone https://github.com/nswebster03/password-strength-analyzer.git
```

Navigate to the project folder:

```bash
cd password-strength-analyzer
```

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Open:

```text
http://127.0.0.1:5000
```
